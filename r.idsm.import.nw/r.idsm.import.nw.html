DESCRIPTION

r.idsm.import.nw downloads image based digital surface model (iDSM, in German bDOM) for Nordrhein-Westfalen (NW) and area of interest. The data "Bildbasiertes Digitales Oberflächenmodell 50" is downloaded from OpenGeodata.NRW and can be used under the specification of the licence: "Datenlizenz Deutschland - Zero - Version 2.0" licence.

EXAMPLE

Import iDSM

Import iDSMs with native resolution: 
r.idsm.import.nw aoi=aoi_NW output=ndsm_NW -r
