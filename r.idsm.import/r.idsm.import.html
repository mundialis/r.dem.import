DESCRIPTION

r.idsm.import downloads and imports image based digital surface models (iDSM, in German bDOM) for specified federal state and area of interest. Implemented federal state options are:
  • Nordrhein-Westfalen (NW)

EXAMPLE

Import iDSM with native resolution: 
r.idsm.import fs=NW aoi=aoi_NW output=idsm_NW -r
