DESCRIPTION

r.dsm.import.bb downloads digital surface model (DSM, in German DOM) for Brandenburg (BB) and area of interest. The data "bildbasiertes Digitales Oberflächenmodell" is downloaded from OpenGeodata.Brandenburg and can be used under the specification of the licence: © GeoBasis-DE/LGB, Datenlizenz Deutschland - Namensnennung - Version 2.0, © GeoBasis Brandenburg/Bildbasiertes Digitales Oberflächenmodell (bDOM).

EXAMPLE

Import DSM

Import DSM with native resolution: 
r.dsm.import.bb aoi=aoi_BB output=dsm_BB -r
