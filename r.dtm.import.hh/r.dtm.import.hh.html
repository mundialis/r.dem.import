DESCRIPTION

r.dtm.import.hh downloads digital terrain models (DTM, in German DGM) for Hamburg (HH) and area of interest. The data "Digitales Höhenmodell Hamburg" is downloaded from daten-hamburg and can be used under the specification of the licence: © GDI-Th, Datenlizenz Deutschland – Namensnennung – Version 2.0, DGM1.

EXAMPLE

Hamburg example

Download and import DTM with native resolution: 
r.dtm.import.hh aoi=aoi output=dtm -r
