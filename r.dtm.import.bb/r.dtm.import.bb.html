DESCRIPTION

r.dtm.import.bb downloads and imports digital terrain models (DTM, in German DGM) for Brandenburg (BB) and area of interest. The data "Digitales Geländemodell" is downloaded from OpenGeodata.Brandenburg and can be used under the specification of the licence: © GeoBasis-DE/LGB, Datenlizenz Deutschland - Namensnennung - Version 2.0, © GeoBasis Brandenburg/Digitales Geländemodell (DGM).

EXAMPLE

Brandenburg example

Download and import DTM with native resolution: 
r.dtm.import.bb aoi=aoi output=dtm -r
