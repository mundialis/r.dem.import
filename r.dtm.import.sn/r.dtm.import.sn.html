DESCRIPTION

r.dtm.import.sn downloads digital terrain models (DTM, in German DGM) for Sachsen and area of interest. The data "Digitales Geländemodell" is downloaded from Landesvermessung Sachsen and can be used under the specification of the licence: GeoSN, Datenlizenz Deutschland – Namensnennung – Version 2.0, DGM.

EXAMPLE

Sachsen example

Download and import DTM with native resolution: 
r.dtm.import.sn aoi=aoi output=dtm -r
